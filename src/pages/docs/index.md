# Table Intelligence docs

Table Intelligence is a **suite of AI agents** for your business — real, defensible work that
plugs into Claude and other AI agents through a **single install**. You install the MCP server
once; every agent in the suite arrives through it.

## The agents

- **[Data analysis](/docs/agents/data-analysis)** — profile, test, model and explain any table.
  Cohorts, RFM, retention, forecasting, classification, association mining and causal inference,
  all running locally on your machine. *(Free)*
- **[Outreach](/docs/agents/outreach)** — your agent researches prospects and drafts personalised
  cold emails; you review and edit on your dashboard, then send from your own email tool. *(Premium)*

More agents are on the way — client research, sales & marketing helpers, connectors to the tools
you already use — and they'll appear in the same place under one subscription.

## Why it's different

Two ideas set these agents apart from asking an LLM to "just analyze this":

- **Deterministic, reproducible math.** Methods are chosen deterministically and run with fixed
  seeds, so the same question gives the same answer — and you can cite it.
- **The honesty seam.** You use these agents *because* you can't run the work yourself — which
  means you also can't check the number. So every result carries a **trust level** and caveats,
  and the agents **decline** rather than invent a number the data can't support.

Your data stays on your machine. We only ever see your API key (and the reports you choose to
save). See [Privacy & data](/docs/privacy).
